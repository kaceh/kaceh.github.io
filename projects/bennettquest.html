My Roles

As the UI/UX Designer, I did the initial observation, user research, low fidelity, high fidelity, prototyping, and testing.

What I've Learned

Observation is a powerful tool if I can use it properly. It helps to reveal some insight that might not be seen through direct confrontation, like interview.

For example, the purpose of this project was found during the observation phase. I recognize that some kids running around the museum when they can actually learn something.

Therefore, my team decided to gamify the museum experience using a treasure hunting game, which still involve walking/running around the museum, but this time with more learning.
